# 🌧️ Rain in Australia — Streamlit + Random Forest

Веб-додаток на **Streamlit**, який прогнозує, чи піде **завтра дощ** в
Австралії, на основі сьогоднішніх погодних спостережень. Модель —
`RandomForestClassifier`, навчена на наборі даних
[weatherAUS](https://www.kaggle.com/jsphyg/weather-dataset-rattle-package)
(≈10 років щоденних спостережень Australian Bureau of Meteorology).

Домашнє завдання до теми **«Дерева прийняття рішень та випадкові ліси»**.

## 🔗 Демо

> Після деплою вставте сюди посилання на застосунок:
> **https://<ваш-застосунок>.streamlit.app**

## 🧠 Як це працює

Дані користувача проходять **той самий препроцесинг**, що й тренувальні
(модуль [`weather_data_processing.py`](weather_data_processing.py)):

1. **Імпутація** пропущених числових значень середнім (`SimpleImputer`);
2. **Масштабування** числових ознак у діапазон [0, 1] (`MinMaxScaler`);
3. **One-hot кодування** категоріальних ознак (`OneHotEncoder`).

Далі підготовлені ознаки подаються в модель Random Forest, яка повертає
прогноз (**Yes/No**) та **ймовірність**. Усі об'єкти препроцесингу
(imputer, scaler, encoder) збережені разом із моделлю у
`aussie_rain.joblib`, тому інференс повністю відтворює навчання.

## 📂 Структура

| Файл | Призначення |
|------|-------------|
| `app.py` | Streamlit-додаток (інтерфейс + інференс) |
| `weather_data_processing.py` | Препроцесинг та функція `predict_input` |
| `train_model.py` | Навчання Random Forest і збереження моделі |
| `aussie_rain.joblib` | Навчена модель + препроцесори (стиснено, ~15 МБ) |
| `requirements.txt` | Залежності |

## ▶️ Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔁 Перенавчання моделі (необов'язково)

Файл `weatherAUS.csv` не входить у репозиторій (він великий і не потрібен
застосунку). Щоб перенавчити модель, покладіть `weatherAUS.csv` поруч і
запустіть:

```bash
python train_model.py
```

> ⚠️ Модель зберігається зі стисненням `compress=('zlib', 3)`, щоб файл був
> менший за 100 МБ і його можна було залити на GitHub.

## 📊 Якість моделі

- Точність на валідаційній вибірці (2015 рік): **~0.86**
